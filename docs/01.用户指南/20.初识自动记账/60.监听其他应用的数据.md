自动记账除了默认支持`微信`和`支付宝`的账单监听，还可以通过监听其他应用的通知来获取账单信息。

::: warning LSPatch用户注意
请确保已授权自动记账的通知使用权限。
:::

::: tip
不建议监听微信，没有太多价值。
:::
